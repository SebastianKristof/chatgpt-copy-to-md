# Store Listing

## Name
ChatGPT Copy as Markdown

## Short description (<=132 chars)
One-click copy from ChatGPT to Markdown. Preserves formatting, code, tables, and math.

## Detailed description
Copy ChatGPT replies and selections as clean Markdown.

Features
• Copy full assistant responses or selected text
• Preserves formatting: headings, lists, blockquotes, code blocks, tables, links, and math
• Lightweight and fast
• Optional enable/disable toggle and shortcut

How it works
The extension runs on chatgpt.com and converts visible response content into Markdown you can paste anywhere.

No account required. No sign-in.

## Permissions justification
- clipboardWrite: needed to copy Markdown to the clipboard
- storage: used to store the enable/disable preference
- chatgpt.com / chat.openai.com: required to inject the copy button and read visible response text
